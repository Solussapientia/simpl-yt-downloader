# Security Policy
# We take security seriously. Please report any security vulnerabilities
# to security@simplyt.com. We will respond within 24 hours.

# Scope
# This security policy applies to:
# - Main application: https://simplyt.com
# - API endpoints: https://simplyt.com/api/*
# - Static assets: https://simplyt.com/static/*

# Bug Bounty
# We currently do not offer monetary rewards, but we will acknowledge
# security researchers who responsibly disclose vulnerabilities.

# Safe Harbor
